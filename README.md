#  Website Bán Trà Sữa
#  Giới thiệu
Đây là dự án website bán trà sữa được xây dựng bằng ASP.NET Core MVC.  
Hệ thống cho phép người dùng xem sản phẩm, đặt hàng và quản lý đơn hàng.
# Công nghệ sử dụng
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML, CSS, Bootstrap
# Chức năng chính
# Người dùng
- Xem danh sách trà sữa
- Xem chi tiết sản phẩm
- Thêm vào giỏ hàng
- Đặt hàng

# Admin
- Quản lý sản phẩm
- Quản lý đơn hàng
- Quản lý khách hàng
 
# Cấu trúc project
- Models: Chứa các lớp dữ liệu
- Views: Giao diện người dùng
- Controllers: Xử lý logic

# Cách chạy project
* Cấu hình Database
- Mở SQL Server
- Mở file `.sln` bằng Visual Studio  
- Chọn Web.config ( ghi chú Data Source= Server Name trên SQL)
- Restore NuGet Packages  
* Chạy project (F5 hoặc Ctrl + F5)
