# Đồ án: Website Bán Trà Sữa
# Giới thiệu
Đây là dự án website bán trà sữa được xây dựng bằng ASP.NET Core MVC.  
Hệ thống cho phép người dùng xem sản phẩm, đặt hàng và quản lý đơn hàng.
# Công nghệ sử dụng
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML, CSS, Bootstrap
# Chức năng chính
# Người dùng
- Xem danh sách trà sữa
- Xem chi tiết sản phẩm
- Thêm vào giỏ hàng
- Đặt hàng
# Admin
- Quản lý sản phẩm
- Quản lý đơn hàng
- Quản lý khách hàng
# Cách chạy  project
1. Mở file `.sln` bằng Visual Studio và SQL (ghi chú: trong Visual Studio khi chạy chương trình trong web.config khi kết nối SQL đổi tên theo server name theo SQL)
2. Restore NuGet Packages  
3. Chạy project (F5 hoặc Ctrl + F5)
